##  Simple Profile Card
    A clean and responsive profile card built using basic HTML and CSS.

##  Purpose of the project
    This project was created to practice fundamentals CSS Concepts such as:
        >Margin and Padding
        >Inline vs Block element
        >Concept of inline-block element
        >Basic layout styling
        >Text alignment

##  Features
    >Profile image with caption
    >Name and Position
    >Short bio section
    >Social media links
    >Clean and simple UI design

##  Built with
    >HTML5
    >CSS3

##  Acknowledgement
    This is a beginner-friendly project built for learning and improving CSS skills.
