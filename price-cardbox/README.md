##  Moonlight Shoes Shop (Practice Project):
    A simple product card website built using HTML and CSS.

##  About the project:
    This project displays a small shoe shop interface with multiple product cards.
    Each card shows a shoe image, description, price, and a "Buy Now" button.
    It is designed as a beginner-friendly project to practice basic web design concepts.

##  Objectives:
    > Practice HTML structure (header, nav, section, footer)
    > Learn how to create reusable product cards
    > Use CSS for styling (backgrounds, borders, spacing)
    > Apply inline-block for layout
    > Add basic interactivity using :hover

##  Features:
    > Navigation menu aligned to the right
    > 3 product cards (ASICS, Hoka, Brooks)
    > Background images for products
    > Styled buttons with hover effect
    > Simple footer with contact details

##  Built With:
    > HTML5
    > CSS3
